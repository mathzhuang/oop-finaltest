# 自选题目：Q 版泡泡堂

## 团队人数：2

### 游戏描述：

本游戏参考经典同名游戏《Q 版泡泡堂》，是一款融合策略、操作与实时对抗的多人爆破对战游戏。游戏支持单人挑战 AI，本地双人合作与联机对战三种模式，采用 Cocos2d-x 框架开发，注重游戏逻辑的清晰实现与模块化设计。

游戏核心玩法围绕玩家操控角色在网格化地图中移动、放置炸弹、摧毁软墙并躲避爆炸展开。特色系统包括：

**地图与隐藏系统：** 支持随机生成地图、隐藏房间与道具、迷雾视野（Fog of War），增强探索与策略性；

**帧同步联机对战：** 采用 Lockstep 同步机制，确保多端操作一致性与低延迟体验；

**智能 AI 系统：** 提供不同难度 AI，具备基础移动、避爆与攻击策略；

**道具与成长系统：** 玩家可通过拾取道具提升移动速度、炸弹数量、爆炸范围与视野范围。

游戏整体架构清晰，涵盖地图模块、玩家控制、炸弹系统、道具系统、AI 逻辑、战斗判定与网络同步等核心模块，适合两人团队分工协作，兼顾游戏性与技术实现深度。

## 系统整体功能设计

**[启动界面]**
- 单人模式（P vs AI）
- 本地双人
- 联机对战（创建房间/加入房间）
- 设置（音量/按键设置）

**[游戏系统核心模块]**
- 地图模块（含隐藏地图、迷雾系统）
- 玩家控制模块
- 炸弹与爆炸模块
- 道具系统
- AI 模块
- 战斗判定模块
- 联机同步模块（帧同步）

## 基础功能：

| 模块         | 功能点                  | 说明                                                                 |
|--------------|-------------------------|----------------------------------------------------------------------|
| 地图系统     | 15×13 或 17×15 网格地图 | 使用 TileMap 实现，包含固定墙、软墙、空地、道具生成点               |
|              | 随机地图生成            | 游戏开始时根据种子（Seed）随机生成地图，主机同步至所有客户端       |
| 玩家控制     | 软墙破坏与道具掉落      | 炸弹可摧毁软墙，随机掉落道具                                         |
|              | 方向移动（上下左右）    | 键盘控制角色移动，支持碰撞检测                                       |
|              | 放置炸弹（初始1个）     | 按下按键放置炸弹，炸弹数量可通过道具提升                             |
| 炸弹系统     | 炸弹倒计时（2秒）与爆炸 | 十字形爆炸范围，可破坏软墙、触发道具、伤害玩家                     |
|              | 爆炸范围与威力提升      | 通过道具可增加爆炸格数                                               |
| 道具系统     | 基础道具类型            | 加速、加炸弹数、加大爆炸范围、扩大视野                               |
|              | 道具拾取与效果生效      | 角色触碰道具后立即生效，持续至本局结束                               |
| 游戏界面     | 开始菜单                | 包含单人、双人、联机、设置选项                                       |
|              | 游戏内UI                | 显示生命值、炸弹数量、倒计时、当前视野范围                         |
|              | 设置界面                | 支持音量调节、按键配置                                               |
| 战斗判定     | 玩家受伤与死亡          | 被炸弹炸中则减少生命值，生命值为0则淘汰                             |
|              | 胜负判定                | 存活到最后一方获胜，或时间结束按剩余生命判定                         |
| 单人模式     | AI 敌人                 | 具备移动、避爆、追击等基础行为                                       |
| 本地双人模式 | 双人间屏对战            | 支持两个玩家在同一设备上进行游戏                                     |

## 附加功能（部分实现）：

|                  | 隐藏房间与传送点          | 部分软墙后藏有隐藏房间，内含强力道具或传送点（彩蛋功能） |

| 道具与成长系统   | 更多道具类型              | 如无敌护盾、穿墙炸弹、遥控炸弹等                 |
|                  | 道具合成或升级系统        | 拾取同类道具可提升效果等级                       |
| 画面与音效       | 角色动画与特效            | 移动、放置炸弹、爆炸等动画效果                   |
|                  | 背景音乐与音效            | 不同场景配乐、爆炸、拾取道具等音效               |
